# Installation :
Open project folder with Android Studio </br>
Build project </br>
Run on device </br>

# Usage :
Enter quick-vision-linux server ip address </br>
Enter port </br>
Enter video frame size (depends on the used neural net) </br>
Enter desired video quality </br>

# Requirements:
Android Studio 2.3  </br>
Android 4.4 ^ </br>
Android NDK  </br>
